# Explore Zambia Website

## Project Overview

Explore Zambia is a multi-page tourism website designed to promote Zambia’s safari destinations and travel services. The website includes a Home page, Services page, and Contact page, all styled using CSS to create a consistent luxury safari theme.

This project demonstrates the use of HTML and CSS to build a structured, visually appealing website with navigation between multiple pages.

---

## Features

- Multi-page website structure
- Navigation bar on all pages
- Hero section on homepage
- Destination cards with images
- Services section with travel offerings
- Contact form layout
- Responsive layout using Flexbox
- Custom color theme inspired by safari environments

---

## Pages Included

- index.html (Home page)
- services.html (Services page)
- contact.html (Contact page)
- style.css (Website styling)

---

## How to Open the Website

1. Download or clone the repository
2. Open the project folder
3. Double-click **index.html**
4. The website will open in your browser
